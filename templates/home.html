Deepfake Detection using Modified Vision Transformer

Developed a robust deepfake detection system using a customized Vision Transformer (ViT) architecture enhanced with Global Attention Module (GAM), Local Attention Module (LAM), and Feature Aggregation Layer (FAL). These custom blocks improved the model's ability to capture both global context and subtle local inconsistencies in manipulated facial regions. Weed Detection using YOLOv8 + BiFPN

Developed an object detection system to accurately identify and localize weeds in agricultural crop images using YOLOv8, integrated with a Bi-directional Feature Pyramid Network (BiFPN) for enhanced multi-scale feature fusion. The system improves detection precision in complex field environments by leveraging both low-level and high-level spatial information. Trained on a labeled agricultural dataset, the model enables early and efficient weed detection, supporting precision farming and reducing the need for manual intervention or excessive pesticide use.
